## ADECART 
 ##### A simple E-Commerce Restful API
### Applications
- Swagger
- Azure Sql Database
- TSQL

### Packages
- Asp.net core Identity
- AutoMapper
- Entity Framework
- Swashbuckle

### Description
##### Adecart API features include:
- Authentication
- Email verification
- Password reset
- PhoneNumber change
- UserName Change
- Order endpoints
- OrderCart endpoints
- User Address endpoints
- Item endpoints
- Payment endpoint
